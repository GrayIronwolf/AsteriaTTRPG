# Phase 3A Character Forge

## Overview
Character Forge replaces the old creator label and becomes the guided character build foundation for Asteria.

## Forge Flow
1. Race
2. Class
3. Appearance
4. Origin
5. Characteristics
6. Skills
7. Equipment
8. Review

## Rules
Class talents are displayed from class progression data and are not freely selected during the forge flow.

Profession slots start as "No Profession Learned" and remain campaign-earned.

Starting skill selection requires exactly 4 skills from the Skill Compendium.

## Compatibility
The older internal create-character route remains as a compatibility alias so existing dashboard buttons, saves, and account routes open Character Forge instead of breaking.
