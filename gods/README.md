# God Database

God entries are treated as religion-domain entries and can be linked from pantheons, factions, locations, and lore.

## Pattern

```text
gods/
  old-iron-saints/
    veloria-the-forge-saint/
      veloria-the-forge-saint.md
```

## Useful Metadata

```yaml
type: god
pantheon:
divine_domain:
alignment:
region:
lore_status:
visibility:
tags:
```
