Unified Workspace

ASTERIA AUTH + WORKSPACE DASHBOARD SYSTEM v1

Firebase Auth

This build keeps the unified compendium and RPG systems intact while adding Firebase account login, persistent sessions, and a logged-in workspace dashboard for campaigns, characters, activity, and account settings.

Locked Foundation

  • v1.3.9.4 account and username login foundation retained.
  • Compendium content folders and manifest system retained.
  • Player dashboard, GM dashboard, inventory/bag UI, dice modal, and talent UI retained.

v1.7.1.3 Session Log Integration

  • Session Log System v1 is integrated into the existing GM Dashboard, not added as a duplicate page.
  • Added states: Not Started, Active, Paused, Ended, Archived.
  • Added Start, Pause, Resume, End, Generate Log, Edit Log, Archive Log, and Export Markdown controls.
  • Automatic log collection now respects session state and only stores session events while Active.
  • Public, GM-only, and Hidden until revealed visibility labels are supported for session events and markdown export.
