# Location Database

Location entries support worlds, realms, planes, regions, settlements, ruins, routes, and encounter areas.

## Pattern

```text
locations/
  realms/
    asteria/
      ironvale/
        ironvale.md
```

## Useful Metadata

```yaml
type: location
location_type:
region:
biome:
climate:
faction:
settlement_type:
visibility:
tags:
```
