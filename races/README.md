# Race Database

Race entries use folder categories for navigation only. Categories do not apply inherited stats or gameplay rules.

## Pattern

```text
races/
  fae-races/
    cavern-sprite/
      cavern-sprite.md
      cavern-sprite-female.png
      cavern-sprite-male.png
      gallery/
```

## Useful Metadata

```yaml
type: race
race_category:
playable:

```
