# Spell Database

Spell entries support spell cards, scaling notes, casting data, source links, and GM-only rulings.

## Pattern

```text
spells/
  arcane/
    light/
      arcane-spark/
        arcane-spark.md
        image.png
```

## Useful Metadata

```yaml
type: spell
spell_school:
magic_type:
element:
rank:
casting_type:
damage_type:
mana_cost:
visibility:
tags:
```
