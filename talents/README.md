# Talent Database

Talent entries are standalone pages that can be linked from class talent trees and character dashboards.

## Pattern

```text
talents/
  martial/
    tier-1/
      guarded-stance/
        guarded-stance.md
```

## Useful Metadata

```yaml
type: talent
class_name:
talent_tier:
rank:
primary_stat:
prerequisites:
cost:
cooldown:
visibility:
tags:
```
